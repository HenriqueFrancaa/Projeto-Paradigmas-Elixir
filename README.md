# Projeto Paradigmas de Programação - Elixir

Este projeto foi desenvolvido como parte da disciplina de Paradigmas de Programação. Trata-se de uma aplicação CLI (Command Line Interface) desenvolvida em **Elixir**, demonstrando conceitos de programação funcional processando entradas do usuário e retornando respostas formatadas.

## Funcionalidades

A aplicação oferece um menu interativo com três funcionalidades principais:

1.  **Conversões Monetárias**: Converte valores de Real (BRL) para Dólar (USD) e Euro (EUR) utilizando taxas de conversão fixas definidas no código.
2.  **Verificação de Dados**: Validação de formato para **CPF** e **E-mail**.
3.  **Estatísticas de Vetor**: Permite ao usuário inserir uma lista numérica e calcula:
    - Média
    - Mediana
    - Valor Mínimo
    - Valor Máximo

## Pré-requisitos

Para executar este projeto, você precisa ter o **Elixir** instalado na sua máquina.

* [Guia de Instalação do Elixir](https://elixir-lang.org/install.html)

Para verificar se está instalado corretamente, rode no terminal:
```bash
elixir -v
```
## Como Executar

1. Clone este repositório: 

```bash
git clone https://github.com/HenriqueFrancaa/Projeto-Paradigmas-Elixir.git
```

2. Navegue até a pasta do projeto:

```bash
cd Projeto-Paradigmas-Elixir
```

3. Execute a aplicação através do arquivo principal:
```bash
elixirc src/*.ex
```
